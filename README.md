# nest-tpl

> nest server template.

[Api Document](./api.md)

## From Docker

```bash
docker pull user/image:tag
docker run -d -p 3000:3000 -it user/image:tag
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
